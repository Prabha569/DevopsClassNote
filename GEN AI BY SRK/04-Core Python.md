Core Python:-
=========================

Comments:-
- Comments can be used to explain Python code.
- Comments can be used to make the code more readable.
- Comments are ignored during execution. (not considered while executing the code)
Single line comment starts with #, and Python will ignore them
  Ex: # this is single line comment

Multi line comments starts ''' & ends with '''
 - Ex:
      '''
       multi
       line
       comment '''

2. Values:-
- Generally values will be of

➤ Numeric
without decimal (Ex: 9, 120, 24000, -20, -400 ) --> int data type
with decimal (Ex: 10.234, 1.02, -20.02 ) --> float data type

➤ Non Numeric (Text)
Only Text  -> String data type , "Krishna" or 'Rama'
combination of text with numeric
combination of text with special characters
➤ Boolean Values
 True  --> boolean data type
 False

Note:- value --> variable--> data type

3. Variables:-
Variable are used for storing values & is a location in memory
In python, variables are Dynamic (We don't even have to declare the type of the variable).
The declaration of data type happens automatically when you assign a value to a variable.

Declaring Variables:-
The equal sign (=) is used to assign values to variables.
      ex: x    =    1
  ->   Variable   Value

Rules in Naming Variables:-

Rule 1: A variable name must start with a letter (a-z, A-Z) or the underscore character (_)
Rule 2: A variable accepts only underscore __ symbol only (No other symbol is accepted)
Rule 3: A variable name cannot start with a number
Rule 4: A variable name can contain combination of alpha-numeric characters and underscores
Rule 5: Variable names are Case Sensitive (age, Age and AGE are three different variables)
Rule 6: It can't accept reserve/keywords
