🧠 1. Arithmetic Operators (math wale)
Operator	Meaning	Example
+	Add	              2+3 = 5
-	Subtract	      5-2 = 3
*	Multiply	      2*3 = 6
/	Divide	              5/2 = 2.5
//	Floor division	      5//2 = 2
%	Modulus (remainder)   5%2 = 1
**	Power	              2**3 = 8

👉 Example:
a = 5
b = 2
print(a+b, a-b, a*b, a/b, a//b, a%b, a**b)

🔍 2. Comparison Operators (compare karte hain)
Operator	Meaning
==	Equal
!=	Not equal
>	Greater
<	Less
>=	Greater equal
<=	Less equal

👉 Example:

print(5 > 3)   # True
print(5 == 5)  # True

⚡ 3. Logical Operators
Operator	Meaning
and	    Dono true hone chahiye
or	    Ek bhi true ho
not	   Ulta kar deta hai

👉 Example:

print(True and False)  # False
print(True or False)   # True
print(not True)        # False
🧩 4. Assignment Operators
Operator	Example
=	    a = 5
+=	    a += 2
-=	    a -= 2
*=	   a *= 2
/=	  a /= 2

👉 Example:

a = 5
a += 2   # a = 7

🧬 5. Identity Operators
Operator	Meaning
is	     Same object
is not	     Different object

👉 Example:

a = [1,2]
b = a
print(a is b)  # True
📦 6. Membership Operators
Operator	Meaning
in	      Present hai
not in	      Present nahi

👉 Example:

print(2 in [1,2,3])  # True
