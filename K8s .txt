=> Pod is a collection of container who share shared storage & shared networking... 

=> Proof = Container to container networking = localhost

=> Pod to pod communication - same namespace

=> pod to pod communication -different namespace

1) how to communicate two container in in a single pod

dual-image-pod.yaml
========================
apiVersion: v1
kind: Pod
metadata:
  name: dual-image-pod
spec:  
  containers:
    - name: nginx-container
      image: nginx
      ports:
      - containerPort: 80

    - name: firefox-container
      image: linuxserver/firefox  
      ports:
      - containerPort: 3000

=> run the pod
$ kubectl apply -f dual-image-pod.yaml

=> port forward
$ kubectl port-forward pod/dual-image-pod 6060:3000

=> In a Single pod two container communicate by using localhost

=> localhost:6060 ( open firefox inside firefox search localhost:80 -- access nginx) 


*)  One pod can connect to another pod using it's private IP ....

add-NP.yaml:-
============
apiVersion: networking.k8s.io/v1
kind: NetworkPolicy
metadata:
  name: villan-np-pod
spec:
  podSelector:
    matchLabels:
      type: tondu
